# KoThok Installer

Download a file, double-click it, plug in your Kobo. Done.

## Windows

1. Download `install.bat`
2. Plug in your Kobo via USB
3. Double-click `install.bat`
4. Follow the on-screen steps

## macOS

1. Download `install.command`
2. Plug in your Kobo via USB
3. Double-click `install.command` in Finder
4. Follow the on-screen steps

## Linux

1. Download `install.sh`
2. Plug in your Kobo via USB
3. Run: `./install.sh`
4. Follow the on-screen steps

## What it does

The installer downloads the latest KoThok binary from GitHub releases and copies
it to your Kobo. First install also sets up NickelMenu so you can launch KoThok
from the device's home screen.

**Requirements:** PowerShell 7 (`pwsh`) and internet connection.
